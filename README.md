# Minesweeper

Minesweeper game built using raylib in c.

## Usage

- Make sure that [raylib](https://github.com/raysan5/raylib) is installed.

```bash
git clone https://github.com/aaronbittel/minesweeper
cd minesweeper
make
./main
```

## Gameplay

- Press `RightClick` or `M` to mark a field as a bomb
- Press `LeftClick` to select a field
- Press `R` to start a new game

## Showcase

### Menu

- choose one of the following difficulties:
    - Beginner: 9x9 (10 mines)
    - Intermediate: 16x16 (40 mines)
    - Expert: 30*16 (99 mines)
